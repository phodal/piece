# Security Policy

## Supported versions

Security fixes are made for the latest released npm package and the current
`main` branch. Before the first stable release, the `0.1.x` line is supported
on a best-effort basis and may receive compatibility fixes together with
security fixes.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Report it
privately through the repository's GitHub security advisory flow:

<https://github.com/phodal/piece/security/advisories/new>

Include the affected version or commit, a minimal reproduction, impact, and
any relevant environment details. Reports involving the npm package, GitHub
Actions workflows, the Gradle wrapper, cache/artifact handling, or language
toolchain invocation are in scope.

We aim to acknowledge reports within seven days and will coordinate a fix and
disclosure timeline with the reporter. Please allow time for a patch before
public disclosure. This project does not offer a bug bounty.
